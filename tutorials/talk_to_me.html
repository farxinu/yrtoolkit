Talk to Me Tutorial

1. Introduction

In this tutorial, you will learn to make an app that speaks words you type in aloud!

2. Add a Button

Under User Interface, drag button to the screen.

3. Change the text on the button to "Talk to Me"

Click on the button to view its settings. "Text" is towards the bottom.

4. Add text-to-speech capabilities

In the Media drawer, drag text-to-speech to the screen.

5. Switch to the blocks editor

Click on the Blocks button in the top right of your screen.

About the blocks editor:

6. Make a button click event

On the left panel, click on button and drag the when Button1.Click block to the workspace.

7. Program the TextToSpeech action

Drag the call TextToSpeech1.Speak block into the when Button1.Click block.

8. Fill in the message socket

Drag an empty text bubble (under Text) and connect it to the message section.

Enter whatever message you want the app to say when the button is clicked

9. Test out your app!

You might have to reconnect the companion app.

10. Shake It

We'll add a feature so the app says something whenever you shake it.

To start, go back to the designer view.

11. Add an accelerometer sensor

The accelerometer sensor is in the sensors drawer

12. Add an accelerometer shaking event

Go back to the blocks editor, and drag a when AccelerometerSensor1.Shaking block into the workspace.

13. Make the app speak when you shake the device

Copy the call TextToSpeech1.Speak block from before and then paste and attach it to the accelerometer block.

Change the text to "Stop Shaking Me!"

Test it out!

14. Say anything

Go back to the designer editor, and drag a TextBox into the screen (it's in the user interface section).

15. Change the hint text

Click on TextBox, and change the hint text to something else

16. Change the message to what the user types

Switch back to blocks editor. Instead of whatever message you had before, drag a TextBox1.Text block instead

17. Congratulations, you made your first app!
