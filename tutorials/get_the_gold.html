Get the Gold Tutorial

1. Introduction

In this game you will be flinging a pirate ship across the screen with your finger!

There will be a coin on the screen that you will be trying to catch with your pirate ship.

If you don't catch it quick enough a timer will cause it to move to a new location.

2. Fling the Pirate Ship

Drag the when PirateSprite.Flung block onto your viewer.

Within the block we will set the heading to the passed-in variable heading. This will make the pirate ship go in the direction that you fling it.

Test your code. Does your pirate ship move when you fling it?

3. Bounce When On Edge

Now we will make the pirate ship bounce when it touches the edge.

Drag out the when PirateSprite.Edge Reached block onto your viewer.

Within this block we will tell the pirate ship to bounce. Drag out the call PirateSprite.Bounce edge block and attach the get edge command.

Test your code. Does the pirate ship bounce when you fling it at an edge.

4. Colliding With the Coin

Now we will write the code for when the pirate ship collides, or touches, the coin. When they touch, the coin should become invisible.

Drag out the when PirateSprite.CollideWith other block along with an if-then statement

For the if part of the statement we will say "if the pirate ship collides with the coin..."

"...then set the coin's visibility to false." Meaning if the coin is touched, make it disappear.

Test your code. Does the coin disappear when the pirate ship touches it?

5. Timer

Here we will be using the when Clock1.Timer for when the timer runs out and we want the coin to move to a new location.

The coin should move to a new random location that is on the canvas. We will use the random integer function.

To make things easier you can duplicate your blocks for the y-position and change the components accordingly:

Test your code. If your pirate ship doesn't touch the coin for a while, does the coin move to a new position?

6. Reset Button

Finally we will set up the reset button so that the coin reappears and the user can play again.

Using the when ResetButton.Click block, set the coinSprite visibility to true. This will make the coin visible again.

Test your code. Does the coin reappear if you press the reset button?
